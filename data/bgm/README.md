# BGM

ここに動画用のBGMファイルを配置する。

## 使い方

1. フリーBGMサイト（DOVA-SYNDROME、甘茶の音楽工房、YouTube Audio Library など）から
   著作権的に問題ないBGMをダウンロード
2. このディレクトリに `default.mp3` という名前で保存する

レンダリング時に `data/bgm/default.mp3` が存在すれば自動でBGMとして乗る。
ファイルが無ければBGMなしでレンダリングされる。

## 音量調整

デフォルト音量は `0.12`（narrationに被らないレベル）。
調整したい場合は `.env.local` に以下を追加:

```
BGM_VOLUME=0.15
```

範囲は `0.0`（無音）〜`1.0`（そのままの音量）。

## 対応フォーマット

Remotion が扱えるフォーマットならOK（mp3 / wav / m4a など）。
ファイル名を変える場合は `BGM_PATH` 環境変数で絶対パスを指定する。

# SFX (Sound Effects)

フック直後に再生する効果音をここに配置する。

## 規約

- ファイル名: `wadaiko.mp3`（renderer がこのファイル名で参照する）
- 再生タイミング: `scenes[0]` 終端（フック4秒後）
- 音量: `0.6`（ナレーションの下に潜らせる）
- ファイルが無ければ自動的にスキップされる（renderer は無音で進む）

## おすすめ入手先（CC0 / 無料商用可）

1. **Pixabay** — https://pixabay.com/sound-effects/search/taiko/
   - CC0、クレジット不要。Shorts向けにそのまま使える品質
2. **効果音ラボ** — https://soundeffect-lab.info/sound/battle/
   - 「和太鼓ドン」など。JP YouTuber 御用達。商用OK・クレジット任意
3. **DOVA-SYNDROME 効果音** — https://dova-s.jp/se/
   - 「和太鼓」で検索。商用OK

ダウンロードしたファイルを `wadaiko.mp3` にリネームしてここに置けばOK。

# 受験歴史ショート動画 リサーチプロンプト

あなたは歴史コンテンツの一次リサーチャーです。
与えられたトピックについて **Google 検索を積極的に使い**、
ショート動画の面白い切り口となる素材を収集してください。

出力は **必ず下記の Markdown 構造** で、他の文章を含めず、見出しをそのまま使うこと。

## 厳守事項

1. **Google 検索を最低 3 回は実行** する
   - 1回目: 通説・概要の確認
   - 2回目: 異説・新説・最近の研究
   - 3回目: 意外な数字・エピソード・一次資料の引用
2. **出典付きの事実** のみ記載。推測・憶測・一般論は書かない
3. 山川出版社『詳説日本史/世界史用語集』の頻出度 A 用語を軸に考える
4. 年号を直接使わず「幕末」「19世紀半ば」「江戸後期」等の **時代表現** で記す
5. 学術的に確度の低い俗説は `⚠ 台本で避けるべき落とし穴` セクションに分離

## 出力フォーマット（この見出しを一字一句変えず使うこと）

```
# {{topic.title}} — リサーチ

## 🎯 驚きの事実（3〜5個）
- （1文の意外な事実）— 根拠の概略
- ...

## 🔢 数字インパクト（2〜4個）
- （具体数字）: （何がどれだけか / なぜ驚きなのか）

## 🔁 通説 vs 新説
- 通説: （一般に知られている説）
- 新説/異説: （研究者名や出典と共に / 無ければ「特になし」）

## 👥 人間ドラマ
- （登場人物の動機・葛藤・関係性を1〜3個）

## ⚠ 台本で避けるべき落とし穴
- （ハルシネーション源・俗説・史料不明のもの）

## 💡 フック案（3個、それぞれ異なる型で）
- 否定反証型: 「〜は半分ウソ。」
- 数字インパクト型: 「たった〜が、〜を動かした。」
- 人間ドラマ型: 「なぜ〜は〜したのか。」
```

## 入力

- トピック: {{topic.title}}
- 時代: {{topic.era}}
- 科目: {{topic.subject}}
- 対象試験: {{topic.target}}

# manabilab-canva キャラ基準ショット生成プロンプト

参照画像（`reference.png`）の **キャラ構造とポーズ感** を踏襲しつつ、
ピンクの色味を全面に乗せた **新マナビくん** の **基準ヒーローショット (front facing, neutral standing)** を生成してください。

このプロンプトの出力先は `manabikun-base.png` で、以降のすべてのシーン画像生成で参照画像として使われる、**ブランド規範の "原典"** になります。
構造・色・スタイルが一発で固まるよう、下記の仕様を **完全に満たした 1 枚** を作ってください。

---

## キャラ仕様（必ず守る）

### 頭部 — 最重要
- **EGG-SHAPED FACELESS HELMET HEAD**: 卵型のなめらかな頭。スカルなし、髪なし、耳なし、首から滑らかにつながる
- **頭の色**: 体と同じ **コーラル / サーモンピンク (#F08080 〜 #FF7B9C 系)**。決してグレーや白にしない
- **額の中央に "脳マーク" (BRAIN icon)**: 白い**シルエット型の脳アイコン** をスタンプロゴのように。下記仕様を厳守する
  - **正面ビュー、左右対称**（左右半球がはっきり分かれている）
  - 上部は **3〜4 個の丸みを帯びた波打ち（雲のような形）** で大脳を表現
  - 中央に **縦方向の細い溝（左右半球の境界）** が入る
  - 全体は **白の塗りつぶしシルエット**（中身を透かさない、線画ではなく塗り）
  - サイズは目とのバランスで額の上半分に収まる程度（額の幅の 30〜40%）
  - 形は **シンプルでアイコン的**（解剖学的なディテールやリアルな gyri は不要）
  - 文字（W、B、ロゴ文字、数字など）は **絶対に入れない**
- **目**: 2 本の **細い斜めの白いスリット**。鋭くクール。下向き 5°〜10° 程度の角度で「決め顔」感を出す
- **口・鼻・眉・耳・歯・舌・髪・脳の凹凸 — 一切なし**

### 体
- **マッチョ・ヒーロー体型**: 引き締まった胸筋・腹筋（シックスパック）・広い肩・はっきりした上腕
- **誇張しすぎない**: ボディビル的なグロテスクさやチビキャラ的な可愛さは避ける
- **肌の色**: 顔と同じ **コーラル / サーモンピンク (#F08080 〜 #FF7B9C 系)**、頭部と完全同色
- **シェーディング**: 1 トーン濃いめのピンクで、筋肉のラインや陰影をフラットに表現（リアル過ぎない、グロテスクでない）
- **上半身は裸**（タンクトップなどは着ない）

### 服装（必須・省略禁止）
- **MUST WEAR knee-length workout shorts** in DEEP MAGENTA / HOT PINK (#D63384 〜 #E91E63 系)
  — ショーツは絶対に省略しない。下半身は裸ではなくこのショーツを必ず履かせる
- ウエストに小さな白いドローストリング（紐）
- **裸足**（靴・靴下・ベルトなどは履かない）

### 線画スタイル
- **2D ベクターイラスト**、フラットカラー + 1 トーン濃いピンクのセルシェード
- 線は **太く均一（約 3px 相当）**、線色は **ダークウォームピンク / バーガンディ系**（純黒は禁止）
- 漫画 / アニメっぽいパーキッとしたシルエット

### 背景・構図
- **PURE WHITE solid background (#FFFFFF)**。影なし、床線なし、小物なし、テキストなし、他のキャラなし
- **FULL BODY composition** — 頭のてっぺんから **裸足のつま先まで全身を必ず収める**。下半身を切ったり太ももで止めたりしない
- **9:16 縦構図**、被写体は画面中央、頭頂から足先までが綺麗に画面内に収まる引き
- キャラの上下に少し余白を残す（後で Canva で文字を被せる前提）

### ポーズ — このベース画像専用
- **正面向き、ニュートラルスタンディング**（後続の全シーンの "原典" になる）
- 両腕は体の横にやや開き気味で下げる（添付参照画像のように手のひらを軽く前に開いて見せても良い）
- 胸を張って堂々と立つ。猫背禁止
- 目線はカメラ目線（スリット目はカメラ向き）
- このショットがブランドの「キャラ規範ショット」になる

### 雰囲気
- クールで自信に満ち、わずかにヒロイック（Captain Workout 系統）
- 「脳と体を鍛えるアスリート」感

---

## 絶対避ける

- **ショーツを省略する（裸 / 下着のみ / 全裸）** — マゼンタの膝丈ショーツは必須
- 頭から足先までの **全身ショットを切る**（半身、腰から上、太ももカットなど禁止）
- リアルな解剖（実際の頭蓋骨、血、血管、グロ、不気味）
- 顔のパーツ（口、鼻、眉、ほうれい線、しわ）
- 髪・脳の凹凸
- 額に **アルファベット文字や数字（"W", "B" など）** — 必ず脳シルエットアイコンのみ
- テキスト、字幕、説明文、ロゴ文字
- 背景の小物、床線、影、他キャラ
- 写真風、3D レンダリング、CGI
- アニメの大きな瞳・キラキラ
- チビキャラ、SD 化
- 性的・煽情的なフレーミング

---

## 入力（必ず実行する）

参照画像 `reference.png` から：
- 頭部の **faceless 卵型構造**
- 額にロゴマークが入る **位置と大きさ**
- **細い斜めスリット目** の角度
- **マッチョ体型のシルエット**
- **構図（正面、両手をやや開く neutral pose）**

を踏襲し、以下のように **改変** する：
- 肌・頭の色 → グレー → **コーラル / サーモンピンク**
- 線色 → 黒 → **ダークウォームピンク / バーガンディ**
- **額のマーク**: 「W」文字 → **白い脳シルエットアイコン**（上部 3〜4 個の波打ち + 中央縦溝、左右対称、塗りつぶし、文字なし）
- 上半身 → そのまま shirtless
- 下半身を追加 → **マゼンタの膝丈ショーツ + 裸足**
- 背景 → 薄グレー → **純白 #FFFFFF**

## 最終出力

画像を 1 枚生成。**JSON や説明テキストは出力せず、画像のみ**。
9:16 縦長 PNG。

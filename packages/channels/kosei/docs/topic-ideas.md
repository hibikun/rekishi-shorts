# 動画トピックリスト（古生物チャンネル）

古生物（恐竜・古代海洋生物・絶滅哺乳類・生きた化石）× スケール逆転 / 常識破壊フックで攻める量産候補。

## 勝ちパターンの型

- **常識破壊型**: 「鳥は恐竜の生き残り」「パンダは肉食獣」など、分類学的な事実で通説をひっくり返す
- **スケール逆転型**: 「ティラノとステゴの時代差 > 人類とティラノ」など、直感を裏切る時間軸/サイズ比較
- **生き抜きの記録型**: 「三葉虫は2億7000万年」「カブトガニは4億年」など、想像を超える持続時間
- **未解明の謎型**: 「ティラノの腕の用途は今も議論中」など、科学が確定していない領域をあえて取り上げる

## 🔥 初期制作候補（量産のための最初の10本）

### [ ] ① 鳥は、絶滅しなかった恐竜である
- **フック**: 系統樹上、ニワトリはティラノサウルスに最も近い現生種
- **勝てる理由**: 小学生でも知ってるが、多くの人は「なんとなく」の理解。系統図で示すと腑に落ちる

### [ ] ② ティラノとステゴサウルスの時間差は、人類とティラノの差より大きい
- **フック**: ステゴ（ジュラ紀後期〜1.5億年前）とティラノ（白亜紀後期〜6600万年前）の間は約8000万年
- **勝てる理由**: サイズ比較図ではなく**時間軸の比較図**が初見殺し。Twitter で定期的にバズる題材

### [ ] ③ メガロドンの歯は、今でもビーチで拾える
- **フック**: 3〜20cmの三角形の歯の化石が、全世界の海岸で発見される
- **勝てる理由**: 身近さ（拾える）+ 巨大さ（歯だけで手のひら超え）のギャップ

### [ ] ④ 三葉虫は、2億7000万年を生き抜いた
- **フック**: カンブリア紀〜ペルム紀末まで。人類（ホモ属）の1000倍以上の持続期間
- **勝てる理由**: 「生き残ってる系（カブトガニ等）」との対比で"今は絶滅"の儚さも演出可能

### [ ] ⑤ カブトガニは、4億年ほとんど姿を変えていない
- **フック**: デヴォン紀からほぼ同じ形。生きた化石の中でも最古クラス
- **勝てる理由**: 今も日本沿岸にいる → 身近 + 4億年の威光

### [ ] ⑥ パンダは、分類学的には肉食獣
- **フック**: ネコ目クマ科。消化器官は肉食用のまま、笹を消化できていない
- **勝てる理由**: 可愛さの裏側。動物園需要で検索流入も期待できる

### [ ] ⑦ カバの最近縁は、ウマではなくクジラ
- **フック**: 偶蹄目クジラ亜目。陸→水の大移行の鍵となる系統
- **勝てる理由**: 直感に反する進化の系統関係。地図/解剖図で絵が持つ

### [ ] ⑧ マンモスは、エジプトのピラミッド建設時代まで生きていた
- **フック**: ウランゲリ島の小型集団が約4000年前まで生存
- **勝てる理由**: 「氷河期の生物」のイメージと「ピラミッド」の同時代性が衝撃

### [ ] ⑨ サメの祖先は、恐竜より1億年以上古い
- **フック**: デヴォン紀から存在。5回の大量絶滅すべてを生き延びた系統
- **勝てる理由**: 現代の海の王者の"本当の経歴"。数字のインパクト大

### [ ] ⑩ アノマロカリスは、最初3種類の別の生物と誤認された
- **フック**: 触手はエビ、口はクラゲ、胴体はナマコ、別個に記載された歴史
- **勝てる理由**: 科学史の意外性 + 変な生物の復元図の絵映え

## 💡 サブ候補（ストック）

### [ ] ティラノサウルスの短い腕の用途は、今も未解明
- **フック**: 繁殖器官説・接触回避説・バランス器官説、決着つかず

### [ ] 地球最大の絶滅は、隕石ではなく火山だった
- **フック**: ペルム紀末大絶滅（P-T境界）、シベリア・トラップの超大規模火山

### [ ] ゴキブリは恐竜より古い
- **フック**: 石炭紀から3億年ほぼ同じ設計

### [ ] クジラの足は、今も骨盤に痕跡として残っている
- **フック**: 進化の証拠としての痕跡器官

### [ ] ダチョウの卵は、ティラノサウルスの卵と同じサイズ帯
- **フック**: 意外にも恐竜の卵は大きくない。鶏卵のほうが異常

### [ ] アンモナイトは、白亜紀末まで3億年間繁栄し続けた
- **フック**: 恐竜と同時に絶滅した頭足類

### [ ] ディメトロドンは恐竜ではない（哺乳類の祖先）
- **フック**: 背中に帆がある有名な古生物。実は単弓類で我々の祖先側

### [ ] 始祖鳥は、本当は"最古の鳥"ではない
- **フック**: 近年の研究で羽毛恐竜の系統に含まれる中間種の一つとみなされつつある

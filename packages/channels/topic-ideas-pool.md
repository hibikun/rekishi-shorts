# 歴史ショート 横断トピックプール (50本)

複数チャンネル (rekishi / ukiyoe / 今後の新規) で使い回せる「教科書レベルの有名人物 × 意外な驚き」型のネタストック。
既存 `packages/channels/{channel}/docs/topic-ideas.md` および `ukiyoe/topic-pool.md` との被りは除外済み (2026-04-26 時点)。

## 驚きパターン凡例

- **A**: 意外な弱点・趣味・性格
- **B**: 教科書知識が実は誤解
- **C**: 意外な人物関係・同時代性
- **D**: 意外な数字・スケール
- **E**: 失敗・凡ミスエピソード

## ステータス凡例

- `[ ]` 未着手 / `[~]` 制作中 / `[✅]` 公開済み
- `⚠裏取り要` = ファクトチェック未完。動画化前に裏取り推奨

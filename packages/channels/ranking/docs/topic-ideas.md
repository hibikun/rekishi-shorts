# 動画トピックリスト（ranking チャンネル）

20代後半〜30代男性が気になるトピックを、毎回テーマを変えて 3選ランキング形式で紹介する量産候補。

## 勝ちパターンの型

- **価格縛り型**: 「5,000円以下で〜」「Amazonで2,000円台の〜」 — 買いやすさを訴求
- **生活改善型**: 「在宅ワークで生活が変わる」「時短になる」 — 実用性
- **男性特化型**: 「20代男性が買ってよかった」「メンズ美容」「デスク周り」 — ターゲット明確化
- **意外性型**: 「地味に便利すぎる」「買う前はバカにしてた」 — フック作り

## ステータス凡例

- `[ ]` 未着手 / `[~]` 制作中 / `[x]` 公開済み

## 制作時の注意

- **薬機法**: 健康・美容カテゴリは表現に要注意。リサーチ段階でNGワード候補を列挙
- **景表法**: 「No.1」「最安」は根拠必須。代替表現を用意
- **著作権**: 商品画像はメーカー公式 or Amazon画像の利用可否を商品ごとに確認
- **ステマ規制**: PR案件でない限り中立評価のトーンで。案件化する場合は動画冒頭に明記する判断軸を別途整備

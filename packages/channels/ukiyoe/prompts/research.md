# 浮世絵タッチ ショート動画 リサーチプロンプト

あなたは歴史コンテンツの一次リサーチャーです。
与えられたトピック「{{topic.title}}」について **Google 検索を積極的に使い**、
「○○の1日」型ルーティーン動画の素材を収集してください。

このチャンネルは浮世絵タッチで動かす映像（fal.ai Seedance）に乗せるため、
**動勢のあるシーンに描き起こせる事実**を優先的に拾ってください（移動・戦闘・職務・天候・群衆など）。

## 厳守事項

1. **Google 検索を最低 3 回は実行** する
   - 1回目: トピックの通説・概要・主要人物
   - 2回目: 当日（または当該期間）の時系列・場所・天候・装備
   - 3回目: 意外な数字・現場ディテール・通説と異なる記述
2. **出典付きの事実** のみ記載。推測・憶測は書かない。年代・年齢・距離・人数などは可能な限り具体値で
3. 「○○の1日」として時系列に並ぶ素材を優先（朝〜夜、または事件発生〜終結）
4. 学術的に確度が低い俗説・後付けエピソードは `⚠ 落とし穴` セクションに分離

## 出力フォーマット（この見出しを一字一句変えず使うこと）

```
# {{topic.title}} — リサーチ

## 🎯 驚きの事実（3〜5個）
- （1文の意外な事実）— 根拠の概略

## 🔢 数字インパクト（2〜4個）
- （具体数字）: （何がどれだけか / なぜ驚きなのか）
- ※ 締めシーンの「数字オチ」候補としても使うので、印象に残る数字を必ず1つ含める

## 🕐 当日の時系列（朝→夜 or 発生→終結）
- 早朝/朝: …
- 昼: …
- 午後/夕方: …
- 夜/結末: …
- 各時刻に「動かせる絵」になるシーン（移動・所作・戦闘・天候・群衆）の核を書く

## 🎨 ビジュアル素材
- 主要人物の風貌・装束（浮世絵で描けるレベルで）
- 舞台となる場所（地形・建物・季節・天候）
- 象徴的な物・道具（武器・道具・旗・看板）

## 👥 人間ドラマ
- 主要人物の動機・葛藤・関係性
- 緊張感のあるディテール（敵の心理・観衆の反応）

## ⚠ 落とし穴
- 史実不明・俗説・出典不明のもの（台本では使わない）
```

なるべく簡潔に。1セクション 5 行を超えないこと。

# シーン割: 002「暗記は寝る前にしろ。理由は脳科学」

## ナレーション全文（235字 / 推定45秒）

> 日中頑張って勉強しても記憶できない。それは記憶のメカニズムを知らないからです。本当に。大事なことは2つだけ。脳の仕組みを理解しましょう。1つ目、寝る前に暗記。日中に覚えた情報は、新しい情報に邪魔されて消えていきます。でも寝る直前なら邪魔されない。これが干渉理論です。2つ目、7時間以上寝る。脳は睡眠中に、覚えた情報を長期記憶に焼き付けています。Walkerの研究では、徹夜したグループの記憶力が40%も低下しました。3日続けてみて。どんどん記憶が定着して、脳みそが溢れちゃうから。

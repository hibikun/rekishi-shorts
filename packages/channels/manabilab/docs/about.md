# 学びラボ (manabilab)

学習科学・認知科学を武器に「**みんなが信じてる勉強の常識**」を1本ずつ壊していく YouTube Shorts チャンネル。

## ポジショニング

> 学習法には「効くもの」と「効かないもの」がある。それは精神論じゃなく、認知科学のエビデンスで決まっている。

Bro Pump がフィットネスで「math と physics で殴る」のと同じ構造を、学習領域に持ち込む。

## チャンネル slug

`manabilab` — `packages/shared/src/channel.ts` 登録済み（subject default: `学習科学`）。

## ターゲット視聴者

| 層 | 規模 | 響くフック |
|---|---|---|
| **受験生**（中高生・浪人） | 大 | 「ノートまとめてる時間、ぜんぶ無駄です」 |
| **社会人学習者**（資格・スキル） | 大 | 「同じ参考書3周読んでも記憶に残らない、本当に効くのは○○」 |
| **学び直し層**（30〜40代） | 中 | 「30過ぎて記憶力が落ちたは嘘。やってる方法が悪いだけ」 |

rekishi（受験歴史）と一部被るが、「**学び方そのものを学ぶ**」メタ層なので食い合わず、むしろ相互送客になる。

## ボイス（語り口）

Bro Pump 直系の **断定 × 法則 × 数字** スタイル。

- **否定フック**: 「Stop X」「Don't Y」 — まず常識を否定して注意を奪う
- **法則アンカー**: 「これは奇跡じゃない、認知科学の法則だ」 と断言
- **数字を1〜2個だけ**: 「24時間で67%忘れる」「想起は受動学習の2倍効率」など強い1個だけ覚えてもらう
- **短文で断定 → 長文で説明** のリズム
- **クロージングは情景**: 「3週間続けてみろ。次の試験で別人になってる」

## 武器（殴れる "法則"）

| 法則 | 一言 | 数字アンカー候補 |
|---|---|---|
| **忘却曲線**（Ebbinghaus） | 24時間で67%忘れる | 67% / 24h |
| **想起練習**（Retrieval Practice） | テストは学習法 | 受動の約2倍 |
| **分散学習**（Spacing Effect） | 集中より分散 | 1日3時間 < 3日1時間 |
| **インターリービング** | 混ぜると伸びる | 同種反復より43%向上 |
| **デザイラブル ディフィカルティ** | 楽な勉強は身につかない | — |
| **テスト効果**（Testing Effect） | 思い出す行為が記憶を作る | — |
| **デュアルコーディング** | 視覚＋言語で記憶定着倍 | — |
| **睡眠と記憶定着** | 寝ないと記憶は固まらない | レム睡眠中の海馬リプレイ |
| **習慣形成**（implementation intention） | 「いつ・どこで・何を」が9割 | 行動成功率2倍 |
| **マインドフルネス・集中** | 注意は筋肉、25分が限界 | Pomodoro の根拠 |

## 想定NG（やらないこと）

- ❌ 精神論（「気合いだ」「努力こそ正義」）
- ❌ 出典のない「俺がやって伸びた」体験談
- ❌ 「東大生が必ずやってる」みたいな権威依存だけのネタ
- ❌ 過度な煽り（「これ知らないとヤバい」連発）
- ❌ 〇〇を1個だけやれば全て解決系（必ず**2〜3の組み合わせ**で出す）

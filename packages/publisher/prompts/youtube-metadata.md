## 出力要件

次の JSON を出力してください。

### title（文字列、40〜60文字を目安・厳守100字以内）
- 視聴者が最初の1秒で「見たい」と思うフック重視
- 数字・疑問形・意外性を1つは入れる
- 末尾に ` #Shorts` を半角スペース区切りで必ず含める（title 全体で100字を超えない）
- 煽り過ぎる表現（「ヤバい」「衝撃！」の乱発）は避け、知的好奇心を刺激する言い回しにする

### description（文字列、400〜800文字・厳守5000字以内）
- 1行目: 動画の内容を1文で要約（title の焼き直しにならないこと）
- 空行
- ### 覚えるべきポイント
  - keyTerms をベースに箇条書きで3〜5項目。受験で狙われる観点を明示
- 空行
- ### こんな人におすすめ
  - 対象試験（{{topic.target}}）に合わせて1〜3行
- 空行
- ハッシュタグ10〜15個を半角スペース区切りで1行に。`#Shorts` `#{{topic.subject}}` は必須

### tags（配列、10〜15要素・各25字以内・合算480字以内）
- 検索されそうな語、別名、人名、時代名、同義語をバランスよく

## 制約
- すべて日本語
- 出力は JSON のみ、前置きや ``` ブロックは不要
- 著作権表記や「画像は AI 生成」等の注釈はシステム側で自動追記するので含めないこと
